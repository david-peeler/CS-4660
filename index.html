Rules

How this match works

  • Each round gives you one challenge about CS hallucinations, prompt surgery, evidence matching, or code verification.
  • Answer quickly to earn more points, but do not trade speed for careless guesses.
  • Hints are available when you need them, but they disappear once you build a 2-question streak.
  • Use the classroom framework to separate helpful support from AI doing the work for you.
  • The leaderboard updates after every round so you can track who is ahead.
